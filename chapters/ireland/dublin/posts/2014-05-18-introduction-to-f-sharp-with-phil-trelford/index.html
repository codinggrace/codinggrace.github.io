Introduction to F# with Phil Trelford
by Vicky Twomey-Lee (Dublin) on 2014-05-27

  • When: Saturday, 12th July 2014 (11:00-16:00)
  • Where: Microsoft Ireland, Building 3, Carmanhall Rd, Sandyford Industrial Estate, Dublin 18 (Map)
  • Price: EUR20.00 (includes lunch)

Phil Trelford will be our F# mentor. The goal of this session is to use F# and sample data from Kaggle to predict who lives and who dies on the Titanic!

To get the most from the session please bring a laptop along with F# installed:

  • Install F# on Windows
  • Install F# on OSX
  • Install F# on Linux

This workshop is a a diversity workshop and all are welcome.

Register: https://ti.to/eventgeekie/fsharp-machine-learning-from-disaster
