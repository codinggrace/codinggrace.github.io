Introduction to Python (Part 3)
by Vicky Twomey-Lee (Dublin) on 2014-01-31

UPDATE: Venue change for Monday evening’s session.

The third part in our journey with Learning Python The Hard Way.

Before we delve any further, we will be going back a few chapters about file I/O and we will start using the PyCharm Editor instead of the awesome iPython Notebook we have been using in the last two workshops.

You have a choice to attend Part 3 on Monday evening or Saturday morning. Only difference is that on Monday evening, it’s a shorter session with no breaks.

MONDAY 10 FEBRUARY 2014 (18:30 - 21:00)

Where: KC Peaches (Upstairs), 54 Dame Street, D2.

Online event registration for [Coding Grace] Introduction to Python Workshop (Part 3) powered by Eventbrite

Saturday 22nd FEBRUARY 2014 (11:00 - 16:00)

Where: Dublin Exchange Building (Beside Georges Dock LUAS stop), George’s Dock, IFSC, D1 (Map)

Sell Tickets Online through Eventbrite

PRE-REQUISITES

  • Python installed on your machine (we are not picky which version) - http://python.org/download/
  • Community Edition of PyCharm - http://www.jetbrains.com/pycharm/download/
  • Completed chapters 1-12, 19, 21-27 - http://learnpythonthehardway.org/book/

Just your laptop (and for those attending on Monday evening, maybe a bite to eat with you as there won’t be a break).

Q. Will there be breaks?

Saturday only, we will have breaks in-between.
