Introduction to Ruby Workshop
by Vicky Twomey-Lee (Dublin) on 2014-02-05

Coding Grace presents our first introduction to Ruby Workshop.
When: Wed 19th Feb 2014 (18:30-21:00)
Where: Dublin Exchange Building (Beside Georges Dock LUAS stop), George’s Dock, IFSC, D1 (Map)

Pre-reqs

Make sure you have Ruby installed.

  • Linux (should be there by default)
  • Mac (it is there by default. v2 in Mavericks, r1.8.7-p358 in Mountain Lion). If it’s not then get installer via http://railsinstaller.org/en
  • Windows (http://rubyinstaller.org/)

For Linux/Mac, if you run the following:

ruby -v

It should come back with something like

ruby 1.9.3p327  (2012-11-10 revision 37606)

If this is returned

-bash: ruby: command not found

Then you need to install Ruby.

Windows

If you haven’t installed Ruby before on Windows, go to http://rubyinstaller.org/.
